== Description ==

= Import your Wordpress.com statistics in your posts meta fields automatically =

This plugin needs the Worpdress.com stats plugin.

Your Wordpress.com statistics account must be activated.

It will automatically import all your view per post stats inside your Wordpress database, using your post's meta information fields.
